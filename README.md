# Lamborghini Sales Dashboard 🚗

An interactive Excel dashboard analyzing Lamborghini sales performance across models, regions, and time.

---

## 📊 Key Metrics
- **Total Number of Vehicles:** 54  
- **Total Sales Volume:** 141,374  
- **Average Vehicle Price:** $374,722  
- **Average Horsepower:** 718.28 HP  
- **Total Revenue:** $43 Billion  

---

## 📈 Dashboard Insights
- **Best Selling Model:** Urus  
- **Highest Revenue Region:** Americas  
- **Most Popular Fuel Type:** Gasoline  

---

## 🔍 Features
- Model distribution analysis  
- Regional sales breakdown (Americas, APAC, EMEA)  
- Yearly sales trend visualization  
- Price distribution insights  
- Fuel type comparison  
- Color popularity trends  

---

## 🛠 Tools Used
- Microsoft Excel  
  - Pivot Tables  
  - Charts & Visualizations  
  - Slicers for interactivity

## 💡 Project Objective
To analyze Lamborghini’s sales data and uncover insights that support strategic decision-making in pricing, production, and regional marketing.

---

## 📊 Business Insights
- SUVs (Urus) dominate overall sales → shift in luxury market preference  
- Americas generate the highest revenue → strong market demand  
- Gasoline vehicles still lead → slow transition to hybrid adoption  
- Mid-to-high price range contributes most to revenue
